SERVICE RESPONSIBILITIES SUMMARY
---------------------------------

1. M.S.N. Peiris (IT23201132)
   - Payment Service   : Integrated Stripe (sandbox) for secure consultation fee payments.
   - AI Symptom Checker: AI-powered symptom input with health suggestions and recommended
                         doctor specialties using an AI/ML API.

2. M.T.C. Peiris (IT23201200)
   - Auth Service           : JWT-based authentication and role management (Patient, Doctor, Admin).
   - Patient Management     : Patient registration, profile management, medical report uploads,
                              medical history, past prescription viewing.
   - Admin Role             : User account management, doctor registration verification,
                              platform operations and financial transaction oversight.

3. W A D A K Appuhamy (IT23479746)

                              consultation completion using third-party services.

4. Dilshan.N (IT23250574)
   - Appointment Service    : Doctor search by specialty, appointment booking, modification,
                              cancellation, and real-time status tracking.
   - Telemedicine Service   : Secure real-time video consultation module integrated with
                              Agora RTC API for patient-doctor video sessions.
